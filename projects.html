NoteCraft - Study Notes Web App

A lightweight study assistant built with Python (Flask), HTML, CSS, and JavaScript designed to help students capture notes, create lists, and organise ideas efficiently.

  • Interactive note-taking with instant add and clear actions.
  • Bullet-list creator for study tasks and revision points.
  • Scribble pad with autosave functionality powered by localStorage.
  • Clean UI styled for readability and simple study workflows.
